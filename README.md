Installation
Clone the repository: GitHub - ZampaglioneGM/ReflexAIChallange

or

npm i @cucumber/cucumber -D

npm i ts-node -D

Clone and npm run dev GitHub - ReflexAI/qa-eng-test: QA Eng test code
